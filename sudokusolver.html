Sudoku Solver

I've made a Sudoku solver using recursive backtracking last summer, but I've picked up Sudoku as a hobby recently and decided to revisit this project. I wanted to make a program that could apply human logic to the board instead of iterative guesses. This Sudoku solver solves the puzzle digit by digit, and tells the user how it deduced each digit. This is a useful tool when trying to complete a hard sudoku, and don't know where to look next.

This program solves 80%+ of the New York Time hard daily sudoku puzzles (their puzzles are computer generated, so sometimes they require brute force), and solves the New York Times easy and medium sudoku puzzles 100% of the time
